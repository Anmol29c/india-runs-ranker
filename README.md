# India-Runs-Ranker 🏆

AI-powered candidate ranking engine built for **India.Runs Track 1** hackathon by Redrob x Hack2Skill.

## What it does
Ranks 1,00,000 candidates for a Senior AI Engineer role using multi-factor scoring algorithm.

## Scoring Factors
- Experience relevance (25 points)
- AI/ML skills matching (30 points)
- GitHub activity score (15 points)
- Job title & summary relevance (15 points)
- Redrob behavioral signals (15 points)

## Tech Stack
Python | JSON | CSV | Multi-factor Scoring Algorithm

## How to Run
```bash
python ranker.py
```

## Output
Generates `ranked_output.csv` with ranked candidate IDs and scores.
